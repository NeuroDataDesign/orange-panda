  • Psych Department at UPenn
  • ICA
    • Temporal (or spatial) ICA decomposition
  • Can use:
    • Infomax
    • FastICA
    • JADE

E Noise

ICA Model

  • Find unmixing matrix (W) so columns of S are as independent as possible
    • S=tcrossprod(X,W)
    • X=tcrossprod(S,M)+E

Whitening

  • Find orthogonal rotation matrix such that the source signal estimates are as independent as possible

Infomax

  • Orthogonal rotation matrix that maximizes joint entropy of a nonlinear function of the estimated source signals

FastICA

  • Orthogonal rotation matrix that maximizes negative entropy of estimated source signal

JADE

  • Orthogonal rotation matrix that diagonals the cumulative array of the source signal

PTak R Package

  • Spatio-Temporal Multiway Data Decomposition Using Principal Tensor Analysis on k-Modes
    • Decompose multi-entries data-array
    • Generalizes singular value decomposition

Built on Core Python Libraries

  • Optional dependencies:
    • Mayavi: 3D rendering
    • Scikit-Learn: decoding tasks
    • Python Data Analysis Library (Pandas): Interfacing with spreadsheet table oriented data processing tools

Standard EEG Analysis Data Pipeline

  • Raw: straight from acquisition system
  • Epochs: segments of raw after each repetition of a stimulus
  • Evoked: averaged epochs

MNE-Python follows same pipeline

  • Raw
    • Stored in instances of Raw class
    • Reads from various file formats
    • Translates to FIF, which organizes info into multi-leaved structure of tags
  • Epochs
    • Also instances, created off of Raw objects
    • Define different vars, such as time window and stimuli
  • Evoked
    • Built from multiple epochs
