Fast Fourier Transform - Library

  • Transform time domain into frequency domain
  • To be plotted in frequency - power spectrum
  • Low-pass filtering with cutoff frequency

wICA - Library

  • Artifact Suppression with Wavelet Enhanced ICA
  • Separation of measured signals into statistically independent components
    • Discard noise artifacts
  • ICA
    • Underestimates Neural Power Spectrum
    • Overestimates coherence between different cortical sides
  • wICA
    • Preserves amplitude and phase characteristics

Empirical Mode Decomposition - Theory

  • Eliminate interference
  • Power interference reduction
  • Decomposes signals into a number of Intrinsic Mode Functions
    • Used to extract instantaneous frequencies
  • Detects / Separates / Removes noise Artifacts
  • Suppresses power interference and movement
    • Preserves a large number of target characteristics from original data

Treatment Prediction of ADHD and Depression - Theory

  • EEG Phenotypes of ADHD Diagnosed Patients
  • EEG Response to Stimulus
    • Changes in Theta / Beta Ratio
  • EEG Phenotypes and Biomarkers for Depression

Finite Impulse Response - Theory

  • Constant Delay over all Frequencies
    • Signal Shape not influenced by Phase Shifts
  • Disadvantages
    • Inefficient / Large Delay
  • Effective for high and low pass filtering

Infinite Impulse Response - Theory

  • Efficient with little delay
  • Disadvantages
    • Output changes because of different phase shifts

Butterworth IIR

  • Online EEG Frequency Analysis
    • Theta / Gamma / Alpha
  • Maximally flat pass/stop bands

Chebyshev IIR

  • Some ripple in passband is allowed
    • Focus is on stop-band behavior and falloff
  • Useful with intermediate frequency ranges

Elliptic IIR

  • Fall-off is critical design criteria
  • Some ripple / phase characteristic degradation is required
