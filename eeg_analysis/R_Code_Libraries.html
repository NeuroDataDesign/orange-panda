E Noise

ICA Model

  • Find unmixing matrix (W) so columns of S are as independent as possible
    • S=tcrossprod(X,W)
    • X=tcrossprod(S,M)+E

Whitening

  • Find orthogonal rotation matrix such that the source signal estimates are as independent as possible

Infomax

  • Orthogonal rotation matrix that maximizes joint entropy of a nonlinear function of the estimated source signals

FastICA

  • Orthogonal rotation matrix that maximizes negative entropy of estimated source signal

JADE

  • Orthogonal rotation matrix that diagonals the cumulative array of the source signal
