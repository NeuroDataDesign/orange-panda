Applications of BCI

  • Medical
  • Entertainment
  • Robotic Control
  • Security

  • Art

Sensory Restoration

  • Auditory
    • Cochlear Implant
  • Visual
    • Retinal Implants
  • Touch
    • Allow paralyzed / amputees to sense objects
  • Taste / Smell
    • Security / Robotics Applications

Motor Restoration

  • Robotic Prosthetics
    • Lower-body prosthetic are more complicated
    • Require balance / stability while brain processes other thoughts

Cognitive Restoration

  • Predict Seizures
    • Stimulate neurons or deliver drugs
    • Prevent spread to other brain regions
  • DBS
    • Parkinson's disease symptoms
    • Relieve chronic pain
    • Record / stimulate memories

Rehabilitation

  • From stroke, surgery, or other neurological conditions

Restoring Communication

  • Ex. Paraplegics / Those with ALS
  • Menus, Cursors, and Spellers

Brain Controlled Wheelchairs

  1. Program Wheelchair with high level commands
    • Go to kitchen, bedroom, etc.
  2. Shared Control
    • Command with highest probability based on surroundings is followed

Web Browser

  • Nessi
    • Boxes around links for navigation
  • Google Earth
    • Change in power of freq. band

Virtual Worlds

  • 3 Binary LDA classifiers
    • Detect if user wants to use device
      • Motor image activity detected
      • Majority vote of group of 3 classifiers was output
    • > 80% accuracy after 5 hours

Google Earth

  • 10 hours of training
  • 20 seconds to navigate to desired country

Other Methods

  • Use evoked potentials
    • ex. P300

Robotic Avatars

  • Assistive robots
    • P300 response
    • 2 commands
      • Which location to go to
      • Which object to pick up
        • User focuses
        • Image in environment flash until P300 is elicited

Robotic Avatars

  • 32 EEG Channels
  • Soft Margin Support Vector Machine Classifier
    • Discriminate btwn. P300 of desired object and response from flashing objects
  • Spatial filters similar to CSP
    • Max distance btwn. means of filtered data (class)
    • Minimize within class variance of filtered data
  • 10 minutes of training
    • 95% accuracy

High Throughput Image Search

  • Sort images likely containing desired object
    • P300 response elicited viewing object
    • Correlate response to image, and move to front of image stack
  • 5 Subjects / 2,500 images
    • Moved 92% of target images
    • To first 10% of sequence

Lie Detection

  • Possession of Guilty Knowledge
    • Memory detection
    • Recognition of:
      • Person / Place / Incident linked to crime
  • Mock case scenario
    • 12.5% of subjects classified as intermediate
    • No false positives / negatives
  • People who commit minor crimes
    • 87.5% correctly classified as innocent or guilty
  • Commercialized system in "Brain Fingerprinting"
